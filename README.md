# End of Line - Web Application

This is an academic web-based implementation of the strategy board game **End of Line**. The project consists of a full-stack application that digitalizes the game's mechanics, allowing for interactive gameplay through a web browser.

The application is built with a decoupled architecture, featuring a dedicated **Backend** (API) and a **Frontend** (Client).
